magenta

magenta is a high level, high level, weakly typed, semi-object-oriented language. it is very early in development as of writing, a list of all current and planned features is available below. documentation and specifications are planned to be extensive with nothing left out, a list of catagorized wiki pages are listed below.

it is designed to be deceptively simple, with a high level C-inspired syntax and readable coding style—but having advanced features as well, such as manual memory management.

#include mypkg as pkg

function main args {
	// loop over a condition array
	[i = 0, i < $(args -> length)] => {
		// output to console
		log "arg #" i " is " $(args[i]);
	}

	return 0;
}
